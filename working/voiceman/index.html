User Voice Manager

October 16, 2023 working

This page is a Javascript implementation of the User Voice Manager.

Click on the number to load an audio file into that slot. Once loaded, the audio can be played, saved (to an audio file) or removed. Once assembled, click on “Save TNW file” at the bottom to save as a user voice file.

This can also be used to load existing TNW files and modify or preview them.
